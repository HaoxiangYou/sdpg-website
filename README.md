# SDPG — Project Page

Project page for **SDPG: Efficient On-policy Visual RL via Stochastic Decoupled Policy Gradient**.

## Local preview

Single static HTML site. Either open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy (GitHub Pages)

1. Push this repo to GitHub (public).
2. Settings → Pages → Source: `main` branch, `/ (root)`.
3. The site will be live at `https://<username>.github.io/<repo-name>`.
